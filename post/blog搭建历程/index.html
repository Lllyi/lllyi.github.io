Blog搭建历程

Dec 19, 2020
1 min read

博客地址 https://lllyi.github.io

2020.12.18

- 已完成

  1. 完成brew源替换。 https://www.jianshu.com/p/b26c7bc14440

  2. 本地安装hugo，选择主题，在post文件下创建一个本地.md文档，能够在本地博客显示。

  3. 在GitHub上创建新仓库，并把hugo搭建的本地博客部署到GitHub上，利用了一些git的基本命令。 https://www.bilibili.com/video/BV1q4411i7gL

- 待做

  1. 学习git基本命令。

  2. 美化博客主页。

  3. 熟练上传博客文档。

  4. 熟悉Markdown文本编辑方式。

- 更新步骤

  1. 到hugo搭建的博客所在文件夹 myblog

  2. 新建一个md文档或者完成对老文档的更新 新建： hugo new post/filename.md

  3. 输入命令 hugo –theme=themename –baseUrl=“https://lllyi.github.io” –buildDrafts

  4. 把博客部署到远端GitHub仓库。

    cd public //切换到之前部署到GitHub上后生成的public文件夹下

    git add .

    git commit -m "yyyy/mm/dd-hh:mm"

    git push -u origin master
